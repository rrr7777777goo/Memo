position
어떤 곳의 위치와 관련된 것
종류 : relative, absolute, fixed, sticky
<div> 기본은 왼쪽 상단
position 속성 기본 값 : relative

absolute : span이 가지고 있는 위치가 부모가 기준점

<div><span></span></div>
이러고 있을 때 div의 위치가 기준점이 된다.

fixed : 화면 기준 위치 설정
fixed는 기존의 다른 것과 겹쳐질 수 있으므로 주의

sticky : fixed과 비슷하면서도 다름
모바일 환경에서 지원됨
fixed처럼 화면 기준으로 위치가 설정되는데 특정 시점에서는 위치가 바뀌지 않고 그대로 고정된다.
